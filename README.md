### 概述

本工程是基于`vue3`搭建的 web 端 2D 地图学习平台，提供较为丰富的地图示例。

### 技术栈

- `openlayers`
- `ol-ext`
- `Typescript`
- `Pinia`

### 平台功能介绍

- **地图控件**
  - [x] 缩放控件
  - [x] 旋转罗盘
  - [x] 鹰眼 
  - [x] 比例尺
  - [x] 全屏

- **功能**
  - [x] 标点
  - [x] 标线
  - [x] 矩形绘制
  - [x] 多边形绘制
